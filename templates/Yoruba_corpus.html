You are welcome to the Yoruba Audio Recording Page

This recorder records wav/PCM audio (48 kHz sampling rate) directly in the browser. It uses Matt Diamond‘s Recorder.js recorder, which is a popular JavaScript library for recording audio in the browser as uncompressed pcm audio in .wav containers.

The purpose of this page is to collect accented Yoruba corpus for the transcripts (available in the Transcript Page ) which are replica of the English corpus collected in English Corpus Page

To fully utilise and get the expected quality recording, kindly take note of the following instructions:

1. Select the range of sentences allocated to you from the transcript page.
2. Ensure your accent is properly taking care of in the recording.
3. Ensure you are cautious of the pace at which you pronounce each word in the sentence.
Do not be too fast and do not be too slow.
4. Ensure your recorded words match that provided in the transcript index chosen. If you mistakenly deviate, kindly reload the page to start over.

5. For saving of audio file, if a text file has tag "YORLEIPZIPPARTA_2_", after recording, to save such file, use the tag name_index no"_YOR"
6. For the instance, if a text file is picked inside a file document "LEIPZIP_YORUBA_TRANSCRIPT_PARTA_2" whose tag is "YORLEIPZIPPARTA_2_", and at index no 18, it is a Yoruba text file, and the Yoruba audio is saved with "YORLEIPZIPPARTA_2_18_YOR.
7. Ensure you select the exact main folder i.e. public health or general domain folder, and that you store the Yoruba audio recorded in the allocated folder particular, paying attention to LEIPZIP_YORUBA_TRANSCRIPT, _PARTA, and _2 for the case of "LEIPZIP_YORUBA_TRANSCRIPT_PARTA_2" as the name of a file document.

8. Upload your recordings using this link Upload Audio Files while simultaneously submit the details of your recordings using the Google form provided on the bottom-right of this page.
9.Record a maximum of 2.5 hours and take a break every 30 minutes of recordings.
10. You can make use of the text areas on the right to store texts.

Submit the following details per audio recording using the bottom-right form:
1. File name you save the recording with.
2. The type of data you just recorded (either general or public health)
3. Duration (in seconds) of each of your recordings
4. Accent type utilised for the recording.

Press the "Record" button to start recording
Press the "Pause" button to pause and press again to resume
Press the "Stop" button to end the recording and download your file.
